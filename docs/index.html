All available Config vars

Config vars are basically the variables which configure or modify bot to function, which are the basic necessities of plugins or code to work. You have to set the proper mandatory vars to make it functional and to start the basic feature of bot.

List of all Available Vars of All-Url-Uploader Bot

  1. API_ID

    Get this value from https://my.telegram.org


  2. API_HASH

    Get this value from https://my.telegram.org


  3. BOT_TOKEN

    Your bot token, create bot & get token from https://t.me/BotFather


  4. BOT_USERNAME

    Your bot username without @


  5. OWNER_ID

    Your Telegram ID, get it from https://t.me/TGIdsBot


  6. LOG_CHANNEL

    your bot log channel id , get it from https://t.me/TGIdsBot. your bot should be add to log channel as admin


  7. WEBHOOK

    If you'r deploy in heroku, leave it as True. else leave it as False
